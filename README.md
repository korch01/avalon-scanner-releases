# Avalon Scanner — загрузка для Windows

**Avalon Scanner** — программа для поиска майнеров Canaan Avalon в локальной сети, просмотра метрик (хэшрейт, температуры, пулы, вентиляторы и т.д.) и массовых операций: LED, смена пула, режим работы, прошивка, перезагрузка.

Этот репозиторий содержит **готовые сборки** для Windows. Исходный код не публикуется.

---

## Системные требования

- **ОС:** Windows 10/11 (64-bit)
- **Сеть:** доступ к майнерам по LAN (API cgminer, обычно порт **4028**)
- Установка не требуется — **портативная** папка с `.exe`

## Установка и запуск

1. Скачайте **avalon-windows-portable-amd64.zip**.
2. Распакуйте в любую папку, например `C:\Tools\AvalonScanner\`.
3. Запустите **`AvalonScanner.exe`** — откроется графический интерфейс.

При первом запуске Windows может показать предупреждение SmartScreen — это нормально для неподписанных сборок; выберите «Подробнее» → «Всё равно выполнить», если доверяете источнику.

## Быстрый старт

1. В программе откройте **«Настройки»** и укажите подсеть, например `192.168.1.0/24` (для одного IP — `192.168.1.100/32`).
2. Нажмите **«Скан»** — в таблице появятся найденные устройства.
3. При необходимости включите **«Мониторинг»** для периодического обновления данных.

**Двойной щелчок по IP** в таблице открывает веб-интерфейс майнера в браузере.

## Обновление программы

В меню **«Справка» → «Проверить обновления…»** приложение сравнивает вашу версию с [version.json](https://raw.githubusercontent.com/korch01/avalon-scanner-releases/main/version.json) и при наличии новой сборки предложит перейти к загрузке.

Чтобы обновиться вручную:

1. Скачайте новый zip из этого репозитория.
2. Закройте Avalon Scanner.
3. Распакуйте архив **поверх** старой папки (замените файлы `.exe`).

---

## Частые вопросы

**Майнеры не находятся**  
Проверьте CIDR, firewall Windows, что майнеры включены и отвечают на порту 4028 (или укажите другой порт в настройках).
